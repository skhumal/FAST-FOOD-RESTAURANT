# Shisanyama Restaurant Website 🍖🔥

A modern, responsive **Shisanyama restaurant website** inspired by South African township (kasi) culture.  
This project focuses on **bold food visuals, weekly specials, and interactive user experience**, built using **HTML, CSS, and vanilla JavaScript**.

The website is designed as a **standalone food brand**, with a long-term vision to integrate into a broader township lifestyle brand ecosystem.

---

## 🎯 Project Purpose

This project was built to:
- Practice real-world front-end development
- Design a campaign-driven homepage (weekly specials like *Beef Week*)
- Create a production-style restaurant website (not a template)
- Showcase clean UI, animations, and interaction logic

---

## 🔥 Key Features

- Responsive navigation with mobile menu
- Hero section promoting **weekly specials**
- Limited-time campaign focus (e.g. *Beef Week*)
- Countdown-style urgency concept
- Specials banner section
- Horizontal food video slider (Instagram/TikTok inspired)
- Hover-to-play food videos
- Smooth scroll animations using ScrollReveal
- Mobile-first and fully responsive layout

---
